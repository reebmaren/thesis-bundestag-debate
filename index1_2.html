Representation of LDA Topic Model output

Among other things, the results of the topic model with MALLET are output in a file (here named "topics30_composition"). This file contains the topic compositions of the documents. In order to evaluate the topic distribution over time, the output file was converted into a Pandas data frame. The variables "speechID", "year" and "faction" were extracted from the metadata of each speech. To convert the size of the data frame into a usable size for visualization, the speeches were grouped by year. The values were summed and then normalized, and the average over 8 years (2 election periods) was calculated. This transformation reduced the size of the data frame from 247457 rows × 34 columns to a size of 66 rows × 31 columns. Data normalization is the process of adjustment and standardization of data values. This eliminates discrepancies that might make it difficult to compare and contrast. The topic proportions are now comparable over 73 years. To visualize the trends of several topics over time, the results were plotted on a line graph.
